# Playboard Live

정적 웹앱 기반의 레크레이션 운영/점수 관리 MVP입니다.

## 목표
- 게임 라이브러리 관리
- 행사 레퍼토리 구성
- 현장 진행 점수 입력
- 관객용 읽기 전용 점수판 제공
- 고정 배점 게임과 퀴즈형 문제당 배점 게임 동시 운영
- 모바일 현장 진행에 맞는 빠른 조작 흐름 제공

## 주요 페이지
- `index.html` — 준비 홈, 세션 생성/선택
- `control.html` — 운영 화면, 모바일 입력 중심
- `display.html` — 관객용 읽기 전용 점수판

## 현재 반영 포인트
- 세션 없이 게임리스트/레퍼토리 편집 가능
- 세션 생성 후 바로 운영 화면으로 이동 가능
- 세션 목록에서 `세션 열기` 버튼 제공
- 진행 탭에 현재 게임 시작 안내 추가
- 모바일 화면에서 버튼/입력/탭 크기 확대
