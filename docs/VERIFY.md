# VERIFY.md

## 기본 검증
- `index.html`, `control.html`, `display.html` 각각 열림
- 게임 10개 이상 샘플 등록 가능
- 레퍼토리 생성 및 순서 변경 가능
- 세션이 없어도 control 화면에서 게임리스트/레퍼토리 편집 가능
- 세션 시작 후 승/패/동점/응원점수를 여러 번 저장해도 게임은 자동 종료되지 않음
- `종료` 또는 `종료 및 한 번 더`를 눌러야 다음 게임으로 넘어감
- 동일 게임 반복 기록 가능
- 총점과 최근 기록이 일치함
- 최근 배점 1건 취소 후 총점/기록이 정확히 되돌아감
- 저장된 기록 1건 수동 수정 후 총점/최근 기록/display 반영이 일치함
- 퀴즈형 게임 1건에서 문제 수/정답 수 기반 점수 계산이 정상 반영됨
- display 화면은 읽기 전용이며 조작 버튼이 없음

## 배포 검증
- Netlify 또는 Vercel 배포 URL 접속 가능
- control 입력 후 display 데이터 반영 확인
- 브라우저 콘솔 에러 없음

## 승인 필요 경계
- 공개 배포
- 도메인 연결
- 토큰 입력/저장 방식 확정
- 유료 서비스 연동
